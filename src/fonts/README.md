# Font Files

This directory should contain the following font files:

- `Oxanium-Regular.woff2`
- `Oxanium-Bold.woff2`
- `Merriweather-Regular.woff2`
- `FiraCode-Regular.woff2`

You can download these fonts from:

- Oxanium: https://fonts.google.com/specimen/Oxanium
- Merriweather: https://fonts.google.com/specimen/Merriweather
- Fira Code: https://fonts.google.com/specimen/Fira+Code

Make sure to download the WOFF2 format for optimal performance.
